# Pizza Sales Analysis Power BI Dashboard
An end-to-end Business Intelligence project built in Power BI to analyze pizza sales data and uncover actionable business insights.

---

## 📌 Project Objective

To transform raw transactional sales data into an interactive dashboard that highlights:

- Customer ordering behavior
- Revenue-driving products
- Sales trends by time, category, and size
- Key performance indicators for decision-making

---

## 🗂️ Data Model

This project follows a star schema model:

- **Fact Table:** order_details
- **Dimension Tables:** orders, pizzas, pizza_types

Relationships were built using primary and foreign keys to enable accurate aggregation and filtering.

---

## 📊 Key KPIs

- Total Sales
- Total Orders
- Top Ordered Pizza
- Top Revenue Pizza

---

## 📈 Insights Generated

- Afternoon is the peak sales period
- Friday & Saturday have the highest orders
- Classic category contributes the most to sales
- Large size pizzas generate the most revenue
- Identified most popular and most profitable pizzas
